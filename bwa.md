BWA
===


Download the last version (bwa-....tar.bz2) from 
<http://sourceforge.net/projects/bio-bwa/files/>

- Uncompress
- Change to the uncompressed directory
- Type `make`. This shoul create an executable file called __bwa__
- Add it to your path
